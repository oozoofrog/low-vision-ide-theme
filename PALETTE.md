# GlareGuard 색상 팔레트

모든 IDE/터미널 테마가 참조하는 공통 색상 정의입니다.

## 설계 원칙

1. **눈부심 억제**: 과도한 고채도/'형광 느낌' 최소화
2. **색상 구분**: 각 색상이 명확히 구분되어야 함
3. **대비 기준**: 기본 텍스트 7:1, 보조 텍스트 4.5:1 이상
4. **의미 이중화**: 색상 + 모양(굵기/밑줄/테두리)으로 의미 전달

## 색상 적용 가이드

### 터미널 (Ghostty, iTerm2 등)
- ANSI 16색 팔레트 직접 사용
- `minimum-contrast = 4.5` 권장

### 에디터 (VS Code, Xcode 등)
- 시맨틱 색상을 토큰에 매핑
- 기본 색상으로 UI 요소 구성

### 색상 선택 우선순위
1. 시맨틱 색상표에서 용도에 맞는 색상 선택
2. 없으면 ANSI 팔레트에서 의미상 가장 가까운 색상 선택
3. 새 색상 추가 시 이 문서에 먼저 정의

## 대비비 참고

배경색 기준 대비비 (WCAG 2.1):

### 다크 테마 (`#0F1115` 배경)
| 색상 | 대비비 | 등급 |
|------|--------|------|

| Red `#FF5C5C` | ~7:1 | AAA |
| Green `#4AD295` | ~9:1 | AAA |
| Yellow `#FFB020` | ~10:1 | AAA |
| Blue `#4AB3FF` | ~8:1 | AAA |

### 라이트 테마 (`#E8E4DF` 배경)
| 색상 | 대비비 | 등급 |
|------|--------|------|
| Foreground `#2D3748` | ~10:1 | AAA |
| Red `#D32F2F` | ~5.5:1 | AA |
| Green `#388E3C` | ~4.5:1 | AA |
| Yellow `#E6A117` | ~3:1 | - (minimum-contrast 적용) |
| Blue `#1565C0` | ~5:1 | AA |

> **참고**: `minimum-contrast = 4.5` 설정 시 낮은 대비 색상이 자동 조정됨
